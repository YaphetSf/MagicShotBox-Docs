MagicShotBox Documentation

MagicShotBox is a precision cinema lens control system that enables precise calibration and control of lens parameters (Focus, Iris, Zoom) directly from your iOS device.

Documentation

Key Features

  • Interactive calibration views for lens rings
  • Numeric and rotary input options for fine-tuning
  • Real-time communication with hardware over Wi-Fi
  • Real-time metadata display
  • Lens library management with JSON import/export
  • Support for both portrait and landscape orientations
  • Professional UI for camera operatives and DITs
